基本NuGet指令

Posted on 2015-08-02 Edited on 2021-02-05 In 工具教學 Views: Disqus:

 

先開啟Package Manager Console

1、查詢可用套件清單

指令為get-package -listavailable -filter jquery

注意，當打入指令按下Enter後會等個好幾秒，查詢清單才會經由網路列出來。

 

2、安裝套件

例如安裝jQuery，其指令為install-package jQuery，

代表替所有新開專案預設改安裝最新jQuery，也替已存在專案安裝最新jQuery

 

也可指定安裝特定版本，指令為install-package jQuery -version 1.10.2

 

3、移除套件

例如移除jQuery，指令為uninstall-package jQuery

如遇到相依性錯誤，可以考慮強制移除

指令為uninstall-package jQuery –force

 

4、一次在多個專案中安裝套件

範例指令為Get-Project -All | Install-Package jQuery -Version 1.10.2

 

5、一次移除多個專案中相同的套件

範例指令為Get-Project -All | Uninstall-Package jQuery -Version 1.10.2

 

6、重新安裝

範例指令為Update-Package -Reinstall jQuery

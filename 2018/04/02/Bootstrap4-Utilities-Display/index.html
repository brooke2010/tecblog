Bootstrap4-Utilities-Display解說

一、格式

<!doctype html>
<html lang="en">
<head>
    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
    <!-- Bootstrap CSS -->
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css" integrity="sha384-Gn5384xqQ1aoWXA+058RXPxPg6fy4IWvTNh0E263XmFcJlSAwiGgFAW/dAiS6JXm" crossorigin="anonymous">

<body>
    <div class="d-sm-inline p-2 bg-success text-white">d-inline</div>

</html>

.d-{breakpoint}-{value}

其中 breakpoint 指的是 xs、sm、md、lg、xl ，

value 指的是 none、inline、inline-block、block、table、table-cell、table-row、flex、inline-flex

例子如下：

.d-sm-inline 表示 sm 以上 ( md、lg、xl ) 的 viewport 寬度，其 display 屬性將被設為 inline。

.d-md-block 表示 md 以上 ( lg、xl ) 的 viewport 寬度，其 display 屬性將被設為 block。

 

但例外的是，.d-xs-{value} 可以被縮寫成 .d-{value} 格式，

.d-inline 表示包含 xs 以上 ( sm、md、lg、xl ) 的 viewport 寬度，其 display 屬性將被設為 inline。

 

二、Hiding elements

由上說明已知 .d-none 表示所有的 breakpoint 皆隱藏，

那 .d-sm-inline 表示 sm 以上皆顯示(為方便說明，已不講究 inline、block… )，

兩者合起來的效果( d-none d-sm-inline )就是只隱藏 xs breakpoint
